Play 24 Game online. Single digits 1-9. Includes all 397 solvable cards (the official single digits 24® Game deck by Suntex only contains 192 cards). There are 404 total possible solvable cards but 7 were removed because these require using non-integer numbers (i.e., fractions, decimals) to get a solution. For example the set [1,3,4,6]. 9s are underlined with red to distinguish them from 6s.

Difficulty ratings were automatically assigned by number of distinct solutions. The more solutions, the less difficult. The less solutions, the more difficult. There are 130 easy, 158 medium, and 109 hard cards.

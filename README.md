Play 24 Game online. Single digits 1-9. Includes all 397 solvable cards (the standard single digits 24® Game deck by Suntex only contains 192 cards). There are 404 total possible solvable cards but 7 were removed because these require using non-integer numbers (i.e., fractions, decimals) to get a solution. For example the set [1,3,4,6].

Difficulties were automatically categorized by number of distinct solutions. It measures how many paths a human has available to stumble onto the answer. The fewer the paths, the harder the card. There are 130 easy, 158 medium, and 109 hard cards.
